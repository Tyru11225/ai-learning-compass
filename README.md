# AI Learning Compass / 人工智能学习指南针

人工智能学习指南针是一个面向中文 AI 小白的开源学习路线规划 Agent。

它帮助用户解决三个问题：

1. 我应该从哪里开始学习 AI？
2. 哪些老师、课程和资源适合我？
3. 我接下来 7 天、30 天、90 天应该怎么学？

## 项目定位

本项目面向中文 AI 初学者，尤其是大学生、AI 应用开发入门者和想用 AI 做项目的人。

它会根据用户的基础、目标、可投入时间和学习偏好，生成个性化 AI 学习路线，并推荐适合的老师、课程、工具和实践项目。

## 核心功能

- 学习画像诊断
- 个性化 AI 学习路线生成
- 老师与课程推荐
- AI 项目实践推荐
- 每周学习任务拆解
- 学习复盘与路线调整

## 第一版 MVP

第一版目标：用户输入自己的学习背景和目标，系统输出一份个性化 AI 学习路线。
