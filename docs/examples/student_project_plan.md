# 示例：大学生 AI 项目学习路线

## 用户输入

我是大二学生，会一点 Python，想学习 AI 做大创项目，每天能学 1 小时，希望 30 天内做出一个作品。

## 一、用户学习画像

- 用户类型：大学生 AI 应用开发初学者
- 当前基础：会一点 Python，但缺少完整 AI 项目经验
- 学习目标：完成一个可展示、可参赛、可写进简历的 AI 项目
- 时间投入：每天 1 小时
- 推荐方向：AI 应用开发入门路线
- 适合路径：Prompt → Python API → RAG 知识库 → 简单 Agent → 项目展示

## 二、当前最适合的学习方向

该用户不适合一开始深入学习复杂数学和底层算法，更适合先从 AI 应用开发入手。

推荐优先学习：

1. AI 基础认知
2. Prompt 基础
3. Python 调用大模型 API
4. RAG 知识库应用
5. 简单 Agent 项目开发
6. 项目文档与 GitHub 展示

## 三、30 天学习路线

### 第 1 周：AI 基础认知与 Prompt 入门

目标：理解 AI、大模型、Prompt、Agent、RAG 的基本概念。

学习内容：

- 什么是人工智能
- 什么是大语言模型
- 什么是 Prompt
- 什么是 AI Agent
- 什么是 RAG 知识库

任务：

- 写出 10 个常用 Prompt
- 用 AI 辅助完成一次资料总结
- 整理 3 个自己感兴趣的 AI 项目方向

阶段产出：

- 一份《AI 学习笔记》
- 一份《AI 项目选题清单》

---

### 第 2 周：Python 与大模型 API 入门

目标：学会用 Python 调用大模型 API，完成最简单的 AI 问答程序。

学习内容：

- Python 基础复习
- API 是什么
- JSON 是什么
- 如何调用大模型接口
- 如何读取用户输入并返回 AI 回复

任务：

- 完成一个命令行 AI 问答 Demo
- 让 AI 根据用户输入生成学习建议
- 将代码上传到 GitHub

阶段产出：

- 一个简单的 AI 问答程序
- 一个 GitHub 仓库初版

---

### 第 3 周：RAG 知识库应用

目标：让 AI 能够基于指定资料回答问题。

学习内容：

- 什么是 RAG
- 什么是向量数据库
- 文档切分与检索
- 本地知识库问答流程

任务：

- 准备一份 AI 学习资料
- 构建一个简单知识库
- 实现“基于资料回答问题”的功能

阶段产出：

- 一个 AI 学习资料问答助手 Demo

---

### 第 4 周：完成 AI 学习指南针 Demo

目标：完成一个可展示的 AI 学习路线规划 Agent。

学习内容：

- 用户画像设计
- 学习路线生成 Prompt
- 资源推荐逻辑
- Streamlit 页面展示
- README 项目说明

任务：

- 设计用户输入表单
- 根据用户输入生成学习路线
- 推荐老师和课程
- 输出 30 天学习计划
- 完成 GitHub README

阶段产出：

- AI 学习指南针 Demo
- GitHub 项目说明文档
- 一份可用于展示或答辩的项目介绍

## 四、推荐老师和资源

### 1. 吴恩达 Andrew Ng

适合阶段：AI 认知入门、机器学习基础  
推荐原因：适合初学者建立 AI 整体认知。

### 2. 李宏毅

适合阶段：机器学习、大模型基础  
推荐原因：中文讲解清晰，适合学生系统理解 AI 原理。

### 3. 李沐

适合阶段：深度学习实践  
推荐原因：适合有一定 Python 基础后，通过代码理解深度学习。

### 4. Hugging Face

适合阶段：大模型应用、Agent、开源模型  
推荐原因：适合学习开源 AI 工具和应用实践。

## 五、第一个 AI 项目建议

项目名称：

AI 学习指南针

项目功能：

1. 输入用户基础、目标和时间
2. 自动生成用户学习画像
3. 推荐适合的 AI 学习方向
4. 生成 30 天学习路线
5. 推荐老师、课程和项目
6. 输出下一步行动清单

项目技术路线：

```text
用户输入
↓
学习画像分析
↓
Prompt 模板生成
↓
调用大模型
↓
生成学习路线
↓
展示结果
